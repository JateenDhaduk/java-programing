# Console Based Banking System

This repository contains a simple Java console application that simulates a basic banking system. It demonstrates object-oriented programming concepts such as inheritance, encapsulation, and polymorphism through `Account`, `Saving`, and `CurrentAccount` classes.

## Project Structure

```
console_based_banking_system/
    Account.java       # Base class for bank accounts
    Saving.java        # Savings account with interest calculation
    CurrentAccount.java# Current account with overdraft
    Bank.java          # Manages accounts and operations
    Main.java          # Entry point for the application
```

## Features

- Create savings and current accounts
- Deposit and withdraw money
- Display account details

## Requirements

- Java Development Kit (JDK) 8 or later

## How to Run

1. Navigate to the project directory:
   ```bash
   cd console_based_banking_system
   ```
2. Compile the Java files:
   ```bash
   javac *.java
   ```
3. Run the application:
   ```bash
   java Main
   ```
